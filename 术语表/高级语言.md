# 高级语言

高级编程语言是一种对计算机细节进行了高度封装的编程语言。与[低级编程语言](/低级语言.md)相比，它更加接近自然语言和数学公式，更易于使用和阅读。

高级语言都需要转换为低级语言才能被机器识别，且一般转换后的程序代码比直接用低级语言设计的程序代码要长，所以运行速度更慢，占用的内存更多。

由于早期电脑业的发展主要在美国，因此一般的高级语言都是以英语为蓝本。
